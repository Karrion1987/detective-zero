
# ADR-0001: Visión de Detective Zero

## Decisión
Iniciar con una API modular en FastAPI para soportar casos, evidencias y cadena de custodia.

## Contexto
Necesitamos un núcleo estable y trazable. Empezamos simple y extensible.

## Consecuencias
- + Extensible por módulos
- + Fácil de contenerizar
- - Panel web se implementará más adelante
